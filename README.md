# React Template (Copier Branch)

This branch contains the Copier template version of react-template.

## Usage

Create a new project from this template:

```bash
copier copy gh:mattwwarren/react-template --vcs-ref copier my-new-project
```

Or with specific options:

```bash
copier copy gh:mattwwarren/react-template --vcs-ref copier my-new-project \
  --data project_name="My Dashboard" \
  --data auth_enabled=true \
  --data auth_provider=ory
```

See `copier.yaml` for all available configuration options.
